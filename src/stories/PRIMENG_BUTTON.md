# PrimeNG Button Story

Esta story demonstra o componente Button do PrimeNG com todas as suas variações e propriedades.

## Como usar

Execute o Storybook:
```bash
npm run storybook
```

Navegue até **PrimeNG > Button** no menu lateral do Storybook.

## Variações disponíveis

- **Primary**: Botão padrão
- **Success**: Botão de sucesso (verde)
- **Info**: Botão de informação (azul)
- **Warning**: Botão de aviso (laranja)
- **Danger**: Botão de perigo (vermelho)
- **Help**: Botão de ajuda (roxo)
- **Secondary**: Botão secundário (cinza)
- **Outlined**: Botão com borda
- **Raised**: Botão elevado
- **Rounded**: Botão arredondado
- **Text**: Botão somente texto
- **WithIcon**: Botão com ícone
- **IconOnly**: Somente ícone
- **IconRight**: Ícone à direita
- **Loading**: Estado de carregamento
- **Disabled**: Estado desabilitado
- **Small**: Tamanho pequeno
- **Large**: Tamanho grande

## Controles interativos

Você pode modificar as seguintes propriedades no painel de controles:

- `label`: Texto do botão
- `icon`: Classe do ícone PrimeIcons (ex: 'pi pi-check')
- `severity`: Variante de cor
- `size`: Tamanho do botão
- `outlined`: Estilo outline
- `raised`: Efeito de elevação
- `rounded`: Bordas arredondadas
- `text`: Estilo texto apenas
- `disabled`: Desabilitar botão
- `loading`: Mostrar estado de carregamento
- `iconPos`: Posição do ícone
